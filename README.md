# GatheRing ䷬

A collaborative multi-agent AI framework built with Python, LangChain, and LangGraph.

## Overview

GatheRing is a highly customizable and modular framework for creating and managing AI agents with complex personalities, diverse competencies, and professional expertise. Agents can collaborate, use external tools, and be configured through an intuitive web interface.

## Key Features

- **Multi-Model Support**: OpenAI, Anthropic, Mistral, and local models via Ollama
- **Complex Agent Personalities**: Modular character blocks, emotional behaviors, competencies
- **Professional Diversity**: Agents can embody any profession or expertise
- **External Tool Integration**: File system, CLI, Git, APIs, databases
- **Web Interface**: Admin and user management (Flask prototype → Django/JS)
- **Fully Tested**: TDD approach with 100% test coverage goal
- **Open Source**: Decentralized and user-focused design

## Quick Start

```bash
# Clone the repository
git clone https://github.com/alkimya/gathering.git
cd gathering

# Install dependencies
pip install -r requirements.txt

# Run tests
pytest
```
